# Uso y variantes del script

Este script está pensado como plantilla reutilizable para practicar
distintos conceptos de automatización.

## 1. Manejo de ficheros
Se puede modificar la función `accion()` para procesar líneas de texto,
analizar contenido o transformar datos.

Ejemplo:
- contar longitud
- convertir a mayúsculas
- analizar formato

## 2. Enumeración básica
Puede utilizarse para verificar servicios accesibles mediante peticiones
HTTP en entornos autorizados.

## 3. Enumeración web
Se puede adaptar para recorrer rutas o archivos y registrar códigos HTTP.

## 4. Pruebas controladas de credenciales
Puede emplearse para simular validaciones en laboratorios, sin ejecutar
ataques reales.

## 5. Identificación de servicios
Puede usarse para recolectar cabeceras HTTP y versiones visibles.

⚠️ Todos los usos están pensados exclusivamente para práctica y laboratorios.
