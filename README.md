# Pentesting Learning Lab

Este repositorio documenta mi proceso de práctica y desarrollo de habilidades
en ciberseguridad y pentesting.

Cuento con conocimientos básicos e intermedios en herramientas y conceptos,
y utilizo este espacio para seguir mejorando mediante práctica en entornos
autorizados.

## Objetivo
- Fortalecer mis habilidades técnicas en seguridad ofensiva
- Aplicar metodologías de pentesting
- Documentar avances y aprendizajes
- Prepararme para oportunidades laborales en ciberseguridad

## Idiomas
- Español: nativo  
- Inglés: nivel intermedio (aprendizaje autodidacta enfocado en lectura técnica)

⚠️ Todo el contenido se realiza únicamente en entornos controlados y legales.
