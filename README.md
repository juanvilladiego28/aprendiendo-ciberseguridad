# Pentesting Learning Repository

Repositorio personal para aprender automatización en Python aplicada a
ciberseguridad y preparación para OSCP.

Este proyecto contiene scripts simples, claros y reutilizables,
pensados para practicar lógica, enumeración y procesamiento de datos.

Utilizo Python como lenguaje principal y herramientas de IA como apoyo
para mejorar estructura, organización y aprendizaje.

El objetivo no es crear herramientas ofensivas completas, sino entender
cómo funcionan internamente y cómo adaptarlas según el objetivo.
